Designer Tips

  • Use a transparent PNG so it blends with any game background.
  • Recommended size: square images, 64x64 to 256x256 pixels.
  • SVG works best, scales without losing sharpness.
  • Once uploaded, switch shape to Custom in the Style tab.
  • You can save the current setup as a Profile for each game you play.

Games where Crosshair F WILL get you in trouble

Do NOT use Crosshair F in these games or modes:

  • Valorant - Vanguard kernel anti-cheat blocks ALL overlays. Game won't even launch with Crosshair F running.
  • Faceit / ESEA tournament matches (CS2) - their anti-cheat flags external overlays
  • PUBG ranked / esports mode - tournament rules ban overlays
  • Fortnite Champion League / FNCS - competitive cup rules may prohibit overlays
  • Any tournament play with prize money - check the rules first

For these games, EXIT Crosshair F from system tray before launching. Casual / ranked matchmaking in most games is fine.

Games where Crosshair F works fine

Tested and confirmed working in casual play:

Fortnite (casual/ranked), Apex Legends, CS2 (matchmaking), Warzone, PUBG (casual), Rust, DayZ, The Finals, Rainbow Six Siege, Overwatch 2, Marvel Rivals, Deadlock, Tarkov (single-player), Hunt: Showdown, Battlefield series, COD MW2/MW3/BO6, Sea of Thieves, Forza Horizon 5, Roblox (with the Roblox preset applied), Minecraft, GTA V, RDR2, Cyberpunk 2077, and any other game running borderless windowed mode.

If your game runs in borderless windowed and doesn't use Vanguard, it works. Period.

Fullscreen mode (must read)

Crosshair F works perfectly in borderless windowed mode. Most modern games support this and it gives the same FPS as exclusive fullscreen.

In exclusive fullscreen, NO overlay can render on top without injecting code into the game, which gets you banned. We do not do that.

Fix: open your game's video settings and switch from "Fullscreen" to "Borderless Windowed" or "Windowed Fullscreen". Zero performance loss on Windows 10/11.

How Crosshair F works (technical)

  • Does NOT inject into game processes
  • Does NOT read game memory
  • Does NOT hook keyboard or mouse input
  • Just a transparent always-on-top window, same category as Discord overlay or your taskbar clock
  • Most anti-cheats (VAC, EAC, BattlEye) do not flag it
  • Vanguard (Valorant) blocks ALL overlays as policy, not because we do anything wrong
